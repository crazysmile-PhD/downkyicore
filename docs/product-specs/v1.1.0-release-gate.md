# v1.1.0 Release Gate

Status: blocked by unfinished integration and architecture work

v1.1.0 是架構版本，不是一般 patch。不得只修改顯示文字、建立 tag 或發布現有 stacked branch。

## 必要功能

- PR #75 的有效 typed-navigation back-stack 行為已在最新架構重作並測試。
- PR #77 的公開收藏夾、失效媒體與空分類修正已在 coordinator/typed navigation 架構重作。
- PR #79 與 #80 的 list URL、列表內搜尋、狀態保留及 deep navigation 已由一個新整合 PR 取代。
- 普通影片、番劇、課程、收藏夾、投稿、列表與下載流程有 deterministic regression tests。
- Bilibili API inventory 已交叉驗證；證據不足的 endpoint 只標風險，不擅自替換。

## 必要架構

- 最終 stacked refactor 正確整合到最新 `main`。
- production source 不含 Prism、DryIoc、EventAggregator、RegionManager、ContainerLocator 或字串 ViewName navigation。
- Domain/Application/Infrastructure/Desktop 的剩餘 boundary exceptions 均在 live plan，沒有被文件誤寫成完成。
- `ai-knowledge-graph.md`、`ARCHITECTURE.md` 與實際 call graph 一致。

## 資料相容

- 現有 settings JSON 可讀、可寫、可 atomic replace。
- SQLite legacy schema 可 migration、reopen、rollback。
- 未完成下載、partial file map、GID、completed segment keys 與 resume state 可跨升級保留。
- release package 不攜帶開發者 cookie、設定、Logs、Cache 或下載資料庫。

## 品質與平台

- strict Release build、完整 tests、format、diff、vulnerability/deprecation audit 全綠。
- Windows x64/x86、Linux x64/arm64、macOS x64/arm64 package validation 全綠。
- Windows、Linux、macOS smoke evidence 來自相同 commit SHA。
- release artifact 包含正確 aria2、ffmpeg、ffprobe 與 checksum manifest。

## 版本唯一來源

通過以上條件後，將 `version.txt` 更新為 `1.1.0`。組件、UI、更新比較、package name、tag 與 GitHub Release 必須從這個來源產生：

```text
Version: 1.1.0
FileVersion: 1.1.0.0
AssemblyVersion: 1.1.0.0
Git tag: v1.1.0
```

發布前應執行 `docs/operations/verification-and-rollback.md` 的 release rehearsal，確認 `main` 工作樹乾淨後才可建立 tag。
