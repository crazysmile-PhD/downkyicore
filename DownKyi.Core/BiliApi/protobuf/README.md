# Protobuf sources

`bilibili/community/service/dm/v1/dm.proto` is the source schema for Bilibili danmaku protobuf payloads.

The checked-in schema is mirrored from `pskdje/bilibili-API-collect`, which archived the upstream `SocialSisterYi/bilibili-API-collect` API collection after the original repository was archived. C# types are generated by `Grpc.Tools` during the normal `dotnet build`; generated `*.cs` output should not be edited or committed manually.

To validate regeneration locally:

```powershell
dotnet restore
dotnet build .\DownKyi.Core\DownKyi.Core.csproj -c Release --no-restore --no-incremental
```
